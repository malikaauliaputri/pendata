3. Ambil contoh threshold split (misalnya):#

  • Threshold = 52 cm

Maka kita akan membagi:

  • Group Left (≤ 52 cm)

  • Group Right (> 52 cm)

4. Hitung Entropy Awal (sebelum split)#

\[ Entropy(D) = -p_1 \log_2(p_1) - p_2 \log_2(p_2) \]
\[ p_1 = \frac{8}{15},\quad p_2 = \frac{7}{15} \]
\[ Entropy(D) = -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} = 0.996 \]

5. Entropy setelah split#

Group Left (≤ 4.6 cm):

  • Data: No 13, 9, 6, 15, 8, 7, 4, 10 → 8 data

  • Kelas: 7 Mawar Cina, 1 Mawar Perancis

\[ Entropy(left) = -\frac{7}{8}\log_2\frac{7}{8} - \frac{1}{8}\log_2\frac{1}{8} \approx 0.543 \]

Group Right (> 4.6 cm):

  • Data: No 12, 3, 1, 14, 2, 11, 5 → 7 data

  • Kelas: 7 Mawar Perancis

\[ \text{Entropy(right)} = -\frac{7}{7} \log_2\left(\frac{7}{7}\right) = -1 \cdot \log_2(1) = 0 \]

Information Gain:

\[ IG = Entropy(D) - \left( \frac{8}{15} \cdot 0.543 + \frac{7}{15} \cdot 0 \right) = 0.996 - (0.2896) = \boxed{0.7064} \]

6. Gini Index#

\[ Gini(D) = 1 - \left(\frac{8}{15}\right)^2 - \left(\frac{7}{15}\right)^2 = 1 - \frac{64}{225} - \frac{49}{225} = 1 - \frac{113}{225} = \frac{112}{225} \approx 0.497 \]

Group Left (≤ 4.6 cm):

  • 7 Mawar cina, 1 Mawar perancis

\[ Gini(\text{left}) = 1 - \left(\frac{7}{8}\right)^2 - \left(\frac{1}{8}\right)^2 = 1 - 0.765625 - 0.015625 = 0.21875 \]

Group Right (> 265):

  • 7 Mawar Perancis

\[ Gini(\text{right}) = 1 - \left(\frac{7}{7}\right)^2 = 1 - 1 = 0 \]

Gini Split:

\[ Gini_{\text{split}} = \frac{8}{15} \cdot 0.21875 + \frac{7}{15} \cdot 0 = \boxed{0.1167} \]

2. Ambil threshold split (contoh):#

Gunakan threshold = 7.6cm (terjadi perpindahan mayoritas dari rosa chinensis ke rosa gallica)

Maka:#

Group Left (≤ 7.6 cm) → Data dengan Diameter Bunga ≤ 7.6 cm

No: 13, 9, 6, 15, 8, 7, 10 → 7 data Kelas: Semua Rosa chinensis

Group Right (> 7.6 cm) → Data dengan Diameter Bunga > 7.6 cm

No: 4, 2, 14, 5, 1, 12, 3, 11 → 8 data Kelas: Semua Rosa gallica

3. Komposisi kelas:#

Sebelum split:

  • 8 Rosa gallica

  • 7 Rosa chinensis

\[ Entropy(D) = -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} \approx 0.996 \]

4. Entropy setelah split:#

Group Left (≤ 7.5 cm):#

  • 7 Rosa chinensis: No 13, 9, 6, 15, 8, 7, 10

  • 1 Rosa gallica: No 4

\[\begin{split} Entropy(left) = -\frac{7}{8} \log_2 \frac{7}{8} - \frac{1}{8} \log_2 \frac{1}{8} \\ = -0.875 \log_2 0.875 - 0.125 \log_2 0.125 \\ \approx -0.875(-0.192) - 0.125(-3) \\ \approx 0.168 + 0.375 = \boxed{0.543} \end{split}\]

Group Right (> 52 cm):#

  • 7 Rosa gallica: No 1, 2, 3, 5, 11, 12, 14

\[ Entropy(right) = -\frac{7}{7} \log_2 \frac{7}{7} = -1 \cdot \log_2 1 = \boxed{0} \]

5. Information Gain#

\[\begin{split} IG = Entropy(D) - \left( \frac{8}{15} \cdot 0.5436 + \frac{7}{15} \cdot 0 \right) \\ = 0.996 - 0.2896 = \boxed{0.706} \end{split}\]

6. Gini Index#

\[\begin{split} Gini(D) = 1 - \left(\frac{8}{15}\right)^2 - \left(\frac{7}{15}\right)^2 \\ = 1 - \frac{64}{225} - \frac{49}{225} \\ = 1 - \frac{113}{225} = \frac{112}{225} \approx \boxed{0.498} \end{split}\]

Gini(Left):#

\[\begin{split} Gini(\text{Left}) = 1 - \left( \frac{7}{8} \right)^2 - \left( \frac{1}{8} \right)^2 \\ = 1 - \frac{49}{64} - \frac{1}{64} = 1 - \frac{50}{64} = \frac{14}{64} = \boxed{0.21875} \end{split}\]

Gini(Right):#

\[ Gini(\text{Right}) = 1 - (1)^2 = \boxed{0} \]

Gini Split:#

\[ Gini_{\text{split}} = \frac{8}{15} \cdot 0.21875 + \frac{7}{15} \cdot 0 = \boxed{0.1167} \]

Visualisasi Decision Tree#

Ini adalah code program untuk mencari decision tree sekaligus rules

import pandas as pd
import matplotlib.pyplot as plt
from sklearn.tree import DecisionTreeClassifier, plot_tree, export_text
from sklearn.preprocessing import LabelEncoder

# === Data Mawar ===
data = {
    "Panjang Tangkai (cm)": [55, 58, 54, 52, 57, 48, 50, 49, 47, 52, 60, 53, 46, 55, 48],
    "Diameter Bunga (cm)": [8.5, 8.2, 8.9, 7.8, 8.4, 6.5, 7.0, 6.8, 6.2, 7.4, 9.2, 8.5, 6.0, 8.2, 6.8],
    "Kelas": [
        "Rosa gallica", "Rosa gallica", "Rosa gallica", "Rosa gallica", "Rosa gallica",
        "Rosa chinensis", "Rosa chinensis", "Rosa chinensis", "Rosa chinensis", "Rosa chinensis",
        "Rosa gallica", "Rosa gallica", "Rosa chinensis", "Rosa gallica", "Rosa chinensis"
    ]
}
df = pd.DataFrame(data)

# === Label Encoding ===
le = LabelEncoder()
df['Label'] = le.fit_transform(df['Kelas'])  # gallica=0, chinensis=1

# === Fitur dan Target ===
X = df[["Panjang Tangkai (cm)", "Diameter Bunga (cm)"]]
y = df["Label"]

# === Train Decision Tree ===
clf = DecisionTreeClassifier(criterion="gini", max_depth=3, random_state=42)
clf.fit(X, y)

# === 1. Cetak Pohon Keputusan ===
print("\n=== Pohon Keputusan (Decision Tree) ===")
tree_rules = export_text(clf, feature_names=list(X.columns))
print(tree_rules)

# === 2. Visualisasi Pohon Keputusan ===
plt.figure(figsize=(10, 6))
plot_tree(clf, feature_names=X.columns, class_names=le.classes_, filled=True, rounded=True)
plt.title("Decision Tree untuk Data Mawar")
plt.show()

=== Pohon Keputusan (Decision Tree) ===
|--- Diameter Bunga (cm) <= 7.60
|   |--- class: 0
|--- Diameter Bunga (cm) >  7.60
|   |--- class: 1

Visualisasi Splitting#

# === 2. Uji 2 Data Baru ===
uji_baru = pd.DataFrame({
    'Panjang Tangkai (cm)': [51, 59],
    'Diameter Bunga (cm)': [7.0, 8.6]
})
prediksi = clf.predict(uji_baru)
hasil = le.inverse_transform(prediksi)

print("\n=== Prediksi 2 Data Baru ===")
for i, (p, d) in enumerate(zip(uji_baru['Panjang Tangkai (cm)'], uji_baru['Diameter Bunga (cm)'])):
    print(f"Data {i+1}: Panjang={p} cm, Diameter={d} cm → Prediksi: {hasil[i]}")

# === 3. Visualisasi Decision Boundary ===
print("\n=== Visualisasi Decision Boundary ===")
plot_colors = "ry"
plot_step = 0.1

x_min, x_max = X['Panjang Tangkai (cm)'].min() - 2, X['Panjang Tangkai (cm)'].max() + 2
y_min, y_max = X['Diameter Bunga (cm)'].min() - 0.5, X['Diameter Bunga (cm)'].max() + 0.5

xx, yy = np.meshgrid(np.arange(x_min, x_max, plot_step),
                     np.arange(y_min, y_max, plot_step))

grid = np.c_[xx.ravel(), yy.ravel()]
grid_df = pd.DataFrame(grid, columns=['Panjang Tangkai (cm)', 'Diameter Bunga (cm)'])
Z = clf.predict(grid_df)
Z = Z.reshape(xx.shape)

plt.figure(figsize=(8, 6))
plt.contourf(xx, yy, Z, cmap=plt.cm.RdYlBu, alpha=0.6)

# Plot titik data training
for label, color in zip([0, 1], plot_colors):
    idx = np.where(y == label)
    plt.scatter(X.iloc[idx]['Panjang Tangkai (cm)'], X.iloc[idx]['Diameter Bunga (cm)'],
                c=color, edgecolor='k', label=le.inverse_transform([label])[0])

# Plot titik uji baru
plt.scatter(uji_baru['Panjang Tangkai (cm)'], uji_baru['Diameter Bunga (cm)'],
            c='black', marker='x', s=100, label='Data Uji')

plt.xlabel("Panjang Tangkai (cm)")
plt.ylabel("Diameter Bunga (cm)")
plt.title("Decision Boundary (Panjang vs Diameter Bunga)")
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

Data 1: Panjang=51 cm, Diameter=7.0 cm → Prediksi: Rosa chinensis
Data 2: Panjang=59 cm, Diameter=8.6 cm → Prediksi: Rosa gallica
